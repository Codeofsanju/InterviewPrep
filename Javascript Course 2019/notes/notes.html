What is the internet? - 8-23-2019

A start: How?

The internet is simply just a long wire. This wire connects a bunch of computers to eachother.

Computers transfer data to eachother through this wire.

Some of these computers have a special job. They are online 24/7 ready to serve you all the data and files you are requesting. These computers are called servers. The computers making requests are called clients.

How Does the Internet Work?

You could imagine a webserver as a giant library. You can go in as a client and request any website you want, and it will serve you everything you need to view the site. You could imagine that having every website ever in this library would back lookup slow. How is this problem solved?

This problem is solved by:

1. You make a request from your browser.

2. Your Web Provider send your request to an DNS server.

3. The DNS (Domain Name System) server maps your website request to an IP address of the website you are trying to access.

4. The DNS Server than sends this ip back to your browser so your browser can now make a direct request to your internet provider for the website you are looking for.

5. The internet provider will than send this request to the Internet Backbone.

6. The Internet Backbone is litterally large wires that connect the internet. Using the IP route we can get to the server that has everything we need to view the website we are requesting.

7. All the files needed to view the website are sent back to our browser to be rendered.

Websites

What are sites made up of?

HTML: Responsible for the structure of a site. If a website was a house, HTML would be the builder.

CSS: Responsible for styling a site. This would be the painters and decorators of a home.

JavaScript: Javascript allows our website to do things or have behavior. This would be a electrician, or plumbing of a house.
